Investment Form Field Values

View and manage all form field values stored in variables

Stored Values

Instructions
  1. Click "Store Field Values" to capture all form field values in JavaScript variables
  2. Click "Display Stored Values" to show the captured values
  3. All values are stored in individual variables with descriptive names
  4. You can use these variables for form validation, submission, or further processing
Tip: Open the browser console to see the variable values being logged when you store them.
Code Example

Here's how the values are stored in variables:

let dtNumber = $("#txtAddDtNumber").val();
let transactionDate = $("#txtAddTrDate").val();
let businessCode = $("#txtAddBssCode").val();
let rm = $("#txtAddRM").val();
// ... and so on for all form fields
